## Requisiti
Per eseguire questo programma è necessario installare Java 21 sul dispositivo.

## Getting Started
Questo programma è una demo, per cui non è prevista alcun tipo di configurazione.

In caso di conflitti con la porta di comunicazione MQTT (default 1833 TCP) è possibile chiudere il programma e modificare con un editor di testo il file settings.json cambiando il numero di porta da voler usare.
